## DMIT 是什么？一句话定位

## 我用了 14 个月，踩过一个坑

第一次下单时，我选了洛杉矶 Lite 套餐，以为"洛杉矶离中国近"——结果延迟稳定在 180ms 以上，高峰期丢包率偶尔飙到 8%。后来才搞清楚，Lite 走的是普通线路，不是 CN2 GIA。

换成洛杉矶 Pro 套餐之后，同一时段延迟降到 130ms 左右，丢包基本消失。这个差距不是玄学，是线路本身的物理差异。

所以选 DMIT 的第一步，不是看价格，是先确认自己需要哪条线路。

## 核心线路说明

**CN2 GIA**：电信骨干直连，去程和回程都走 GIA，延迟低、稳定性强，适合对速度敏感的业务。

**CMIN2**：移动国际精品网络，对联通和移动用户更友好，部分场景下表现优于 CN2 GIA。

**Premium Network**：DMIT 自有优化线路，混合多运营商，三网均衡，适合用户分布分散的场景。

**Lite / Eyeball**：普通线路，价格低，适合对回国速度没有强需求的用户（比如纯海外业务）。

## 三个机房怎么选

**优先香港**：你的用户主要在中国大陆，且预算允许。30–50ms 的延迟是其他机房给不了的。

**选日本**：移动用户占比高，或者业务同时覆盖日本本地市场。TYO.Pro 的 CMIN2 对移动用户非常友好。

**选洛杉矶**：用户分布在北美，或者需要大流量低成本方案（Eyeball 系列 1TB 起步，价格只有 $6.9）。如果有回国需求，一定选 Pro 系列，不要选 Eyeball。

## 新手操作步骤：从注册到开机

1. 访问 DMIT 官网，点击右上角 Register 完成注册，填写真实邮箱（账单和通知都走这里）。
2. 进入 Order 页面，按机房 → 线路 → 配置的顺序筛选套餐。
3. 选择计费周期：月付灵活，年付通常有 15–20% 折扣，具体以结账页面显示为准。
4. 完成支付后，系统自动开机，通常 5分钟内收到 IP 和 root 密码邮件。
5. 用 SSH 连接，建议第一步修改默认 root 密码，第二步配置防火墙规则。
6. 如果开机后发现延迟异常，先用 `traceroute` 确认路由路径，再决定是否换套餐或换机房。

## FAQ

**Q：DMIT 的 IP 会被墙吗？**
DMIT 使用的 IP 段整体洁净度较高，但没有任何服务商能保证 IP 永不被封。香港和日本节点因为线路质量好、用户群体相对专业，IP 被封的概率低于普通 VPS。如果遇到 IP 问题，可以通过工单申请更换，部分套餐支持付费换 IP。

**Q：和搬瓦工比，DMIT 哪个更好？**
两者都走 CN2 GIA，定位接近。DMIT 在香港和日本节点上有明显优势，延迟更低；搬瓦工在洛杉矶的套餐选择更多，价格梯度更细。如果你的核心需求是亚太低延迟，DMIT 更合适；如果主要跑美国业务，两者差距不大。

**Q：退款怎么操作？**
登录后台 → 进入 Support → 提交工单，说明订单号和退款原因。72 小时内的新用户退款基本秒批，超出窗口期的情况需要客服人工审核。退款原路返回，PayPal 通常 1–3 个工作日到账，信用卡 3–7 个工作日。

**Q：小白能用吗？没有 Linux 基础怎么办？**
DMIT 提供标准 Linux 镜像（Ubuntu、Debian、CentOS），没有图形化面板。如果完全没有 Linux 基础，建议先学会基本的 SSH 连接和命令行操作，或者搭配宝塔面板使用。DMIT 本身不提供托管服务，技术问题需要自己解决或通过工单咨询。
